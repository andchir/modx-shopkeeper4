Shopping Cart snippet
=====================

A universal snippet for creating a shopping cart in an online store. Also, a snippet
can be used to temporarily store any data, for example, favorites functionality, etc.
Data is saved to the database, you can configure the storage time. You can control the
addition of goods to the basket and edit the contents using JavaScript
functions (without dependencies).
